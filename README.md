# Machine Learning Curriculum for Data Science Masters

Welcome to your personalized machine learning curriculum! This learning path is designed to take you from foundational concepts to advanced machine learning techniques, with a focus on efficient and performant code using NumPy and native Python instead of Pandas.

## Development Environment

### Cursor IDE
We'll be using Cursor IDE for this curriculum. Cursor is a modern, AI-powered IDE that enhances your coding experience. Check out the `resources/cursor_guide.md` for:
- Essential shortcuts and features
- Best practices for ML development
- AI assistance tips
- Debugging techniques

## Curriculum Structure

### 1. Foundations (2-3 weeks)
- Python Programming Review
- Mathematics for Machine Learning
  - Linear Algebra
  - Calculus
  - Probability & Statistics
- Data Manipulation & Analysis
  - NumPy for Data Processing
  - Pure Python Data Structures
  - Data Visualization (Matplotlib, Seaborn)
  - Efficient Data Processing without Pandas

### 2. Machine Learning Basics (4-5 weeks)
- Supervised Learning
  - Linear Regression
  - Logistic Regression
  - Decision Trees
  - Random Forests
  - Support Vector Machines
- Model Evaluation & Validation
  - Cross-validation
  - Metrics (accuracy, precision, recall, F1-score)
  - Bias-Variance Tradeoff

### 3. Advanced Machine Learning (4-5 weeks)
- Unsupervised Learning
  - Clustering (K-means, Hierarchical)
  - Dimensionality Reduction (PCA, t-SNE)
- Ensemble Methods
  - Bagging
  - Boosting (AdaBoost, XGBoost)
- Feature Engineering & Selection

### 4. Deep Learning (6-7 weeks)
- Neural Networks Fundamentals
- Deep Learning Frameworks (PyTorch/TensorFlow)
- Convolutional Neural Networks (CNNs)
- Recurrent Neural Networks (RNNs)
- Transformers & Attention Mechanisms

### 5. Large Language Models & Modern AI (4-5 weeks)
- Transformer Architecture Deep Dive
  - Self-attention Mechanisms
  - Multi-head Attention
  - Position Embeddings
- LLM Fundamentals
  - Pre-training and Fine-tuning
  - Prompt Engineering
  - Context Length and Token Management
- Practical LLM Development
  - Using AI Suite
  - Model Deployment and API Integration
  - Efficient Inference Techniques
- Advanced LLM Topics
  - RAG (Retrieval Augmented Generation)
  - Fine-tuning Strategies
  - Model Quantization
  - Ethical Considerations
